# 关于保研

保研是升学的重要途径之一，西工大各学院的保研率在20%-30%之间。

对于计算机学院而言，保研率在20%-25%之间，主要有以下三种保研方式：

1. 学习成绩保研

2. 综合测评保研

3. 其他保研
